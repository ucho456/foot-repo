# フットレポ

誰でも気軽に日本代表戦や W 杯、欧州 5 大リーグ、CL などの選手採点を投稿したり閲覧する事ができます。
試合を通じて感じた事を選手採点という形に変えて世界にシェアしましょう！

リンク [フットレポ](https://foot-repo.com/)

## 使用技術

Nuxt.js 2.15.8 (SSR, Composition-api)
TypeScript
Vuetify 2.5.5
Firebase 9.6.7
Authentication
Firestore
Cloud Storage
Cloud Functions
Cloud Run
Docker
Github Actions
