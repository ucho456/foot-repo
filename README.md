# フットレポ
サッカーの選手採点を投稿しTwitterで記事をシェアできるサービスです。

<br />

## リンク

<s>https://foot-repo.com/</s>

サーバー代やAPI代で赤字なので現在サービスを停止しました。

<br />

## 使用技術一覧

- Nuxt.js v2.15.8
- TypeScript es2018
- Vuetify v2.5.5
- Firebase v9.6.7
  - Authentication
  - Firestore
  - Cloud Storage
  - Cloud Functions
  - Cloud Run
- Docker v20.5.2
- Github Actions
